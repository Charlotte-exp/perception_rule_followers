Stage 1: Die rolls

  • You will roll a die {{ C.NUM_ROUNDS }} times.
  • After each roll, you report the number you rolled.
  • The number you report is the number of points you will be paid for that roll.
  • 1 point is worth {{ C.conversion }}.

Stage 2: Interaction with another participant in the study

{{ if player.treatment == 'TG' }}
  • After the {{ C.NUM_ROUNDS }} die rolls, you are matched with next participant from this same study.
  • First, you will see the number of times the next participant reported the correct number.
  • Based on this information, you will have the opportunity to send points to them.
  • Every point you send is tripled: 1 point sent becomes 3 points received.
  • Second, the next participant then decides how many points from those, if any, to return to you.
  • They can return nothing, some, or all points.
Example:
  • If you send all 3 points, the next participant will receive 9 points.
  • If they return half of the 9 points, you and the next participant will both earn 4.5 points, and will both have more than you started with.
  • If they return nothing, they will earn 9 points, while you will earn nothing.

Your bonus will be the points the next participant returns to you, and the bonus of the next participant will be the points they keep. You will also decide how much to return to the previous participant who had the same opportunity to send you points based on the number of times you reported the correct number.

{{ endif }} {{ if player.treatment == 'DG' }}
  • After the {{ C.NUM_ROUNDS }} die rolls, you are matched with the next participant from this same study.
  • First, you will see the number of times the next participant reported the correct number.
  • Based on this information, you will interact in a two stage task with them.
  • You will the opportunity to earn another bonus based on that interaction ranging from 0 to 9 points.
Example:
  • If you send all 3 points, the other participant will receive 9 points.
  • If they return half of the 9 points, you and the other participant will both earn 4.5 points, and will both have more than you started with.
  • If they return nothing, they will earn 9 points, while you will earn nothing.

You will also decide how much to return to the previous participant who had the same opportunity to send you points based on the number of times you reported the correct number. This will be ANOTHER PARTICIPANT than the one you decide to send points to.

{{ endif }} {{ if player.treatment == 'rating' }}
  • After the {{ C.NUM_ROUNDS }} die rolls, you are matched with the next participant from this same study.
  • First, you will see the number of times the next participant reported the correct number.
  • Based on this information, you will rate how trustworthy you find them.
  • You will be paid a {{ C.bonus }} points bonus if your rating is the same as the most common answer for that participant.
{{ endif }}
